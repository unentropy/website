# Unentropy Website 

Website for [Unentropy](https://github.com/unentropy/unentropy), a CI/CD tool for tracking code metrics.

**Live site:** https://unentropy.dev

## Development

```bash
bun install    # Install dependencies
bun dev        # Dev server at localhost:4321
bun build      # Build to ./dist/
```

## Project Structure

```
src/content/docs/   # Markdown documentation
src/pages/          # Root pages
src/layouts/        # Astro layouts
src/styles/         # Global CSS
```
